# REVERIE — 品牌规范 v0.1

> 流动式实验艺术机构 / Nomadic Experiential Art Institution
> 自 2018 年起；32 位 Patrons；不复返作品

## 一句话定位

REVERIE 在野外、洞穴、雪线、盐漠现场制作只发生一次的实验艺术作品；每一件作品向 32 位 Patrons 开放、向公众闭门。

## 声音 / Voice

- **克制 · 不解释** — 不写"创新"、"打破"、"重新定义"
- **不引用流量数字** — 不出现 "10×"、"99%"、"百万"
- **物理在场** — 写温度、湿度、声压、火焰；不写"沉浸式体验"
- **承认局限** — "只发生一次"、"不复返"、"不录制"
- **中英并置** — 中文先、英文后；英文是中文的回声而非翻译

不允许出现的词：immersive、unforgettable、journey、game-changing、redefine、unleash、curate

## 色彩 — 与 Mythic Naturecore 锁定一致

```
--ink           #0a0608   近黑底（页面 base）
--earth         #3b1a0a   暖深棕（移动端 Hero 文字）
--paper         #f3ede4   冷象牙（罕用，仅作长篇文字背板）
--gold          #c5a472   暖金（仅用于强调语，每屏 ≤ 2 处）
--mist          rgba(255,255,255,.62)  说明文字
--rule          rgba(255,255,255,.12)  分割线
```

霓虹紫、青色科技色、Material 强彩、亮蓝 CTA — 严禁。

## 字体栈

```
--display: 'Glendora à', 'Noto Serif SC', serif      /* 大标题 */
--body:    'Imprima',     'Noto Sans SC',  sans-serif  /* 正文 */
--mono:    'JetBrains Mono', ui-monospace, monospace   /* 元数据 */
```

中文衬线落到 Noto Serif SC 500，无衬线落到 Noto Sans SC 300。中英并置时中文略大、英文略缩、字距扩 0.06em。

## 排版纪律

- 一屏一意象，一屏一句标题
- 段落 ≤ 3 行；中文行长 ≤ 24 字、英文 ≤ 56 ch
- 标题永远用 Glendora à / Noto Serif 衬线，导航与标签永远用无衬线
- 一个强调色每屏不超过 2 次
- 不在标题边放图标；不给段落加左色条
- 编号用罗马数字（I · II · III）或大写英文字母

## 版面节拍

```
Hero        100vh   一句标题 + 一句副标 + 一行素描，仅此而已
长文章节    auto    左 1/3 元数据竖排，右 2/3 双语正文
图条        100vh   3 张同尺寸静帧并排，无文字
入会区      80vh    单一 CTA + 32-Patron 数字
```

## 装饰预算

每页只允许一处装饰性闪光：
- 首页：窗帘开 + 门户穿越（已用满）
- World 列表：图条交错式入场
- 单 World 详情：一段倒数（"距闭幕还剩 N 夜"）
- 入会页：一处暗金强调线

超出此预算的装饰一律拒绝。
